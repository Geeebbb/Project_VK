### Цель проекта:
Разработать веб-сервис (API), который хранит данные пользователей и сегментов, а также позволяет управлять этими сегментами: необходимо реализовать функции создания, удаления и редактирования сегментов, а также распределение сегментов случайным образом на определённый процент пользователей.
Задачи проекта:
1.	Создание базы данных:
Реализовать таблицы баз данных, создать связи.
2.	Реализация API:
Реализовать эндпоинт для создания, изменения и удаления сегментов, эндпоинт для добавления и удаления пользователей в сегментах, эндпоинт для получения списка сегментов по user_id, эндпоинт для распределения сегмента случайным образом на заданный процент пользователей.
3.	Тестирование сервиса:

Провести тестирование программы.
Краткое описание проекта:
Проект представляет собой веб-API, написанный на Python с использованием FastAPI.
Данные хранятся локально, в файле test.db.
Через запросы к сервису по сети сервис позволяет управлять сегментами и пользователями, а также автоматически распределять сегменты между случайно выбранными пользователями в заданной доле.
